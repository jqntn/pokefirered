# Native Port

Current native desktop work lives in this directory.

## Controls

These controls apply to the current desktop build for now.

### Keyboard

- Arrow keys: D-Pad
- `X`: A
- `C`: B
- `Enter`: Start
- `Right Shift`: Select
- `S`: L
- `D`: R

### Gamepad

- D-Pad: D-Pad
- South / bottom face button: A
- East / right face button: B
- Start / Menu: Start
- Back / Select: Select
- Left trigger 2: L
- Right trigger 2: R

## Notes

- The native build currently boots to the title screen / main menu flow by default.
- `--demo` starts the PAL demo path instead, using the same controls.
- Control mappings may still change as the port matures.
